# Team Wiki Vault

이 저장소는 Obsidian + Claude Code + Wiki Basic Model 협업을 위한 중앙 지식 베이스입니다.

## 구조
- 00-Home.md: 메인 Wiki 홈페이지
- 01-Projects/: 프로젝트별 문서
- 02-Knowledge/: 지식 베이스
- 03-Meetings/: 미팅 노트
- 04-Code/: 코드 문서 및 스니펫
- 05-Templates/: 템플릿 모음

## 사용법
1. Obsidian에서 이 폴더를 Vault로 엽니다.
2. Obsidian Git 플러그인으로 GitHub와 동기화합니다.
3. Claude Code와 연동하여 AI 협업을 시작하세요.
