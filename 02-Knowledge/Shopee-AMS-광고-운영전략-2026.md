## 1. Shopee AMS 개요

**Shopee AMS (Advertising & Marketing Solutions)** 는 Shopee 플랫폼 내에서 셀러가 상품과 샵을 노출시키고 매출을 증대시키기 위한 광고 솔루션이다. CPC(클릭당 비용) 기반의 퍼포먼스 광고부터 CPM(노출당 비용) 기반의 디스플레이 광고까지 다양한 옵션을 제공한다.

### AMS 광고의 핵심 특징

| 특징 | 설명 |
|:-----|:------|
| **과금 방식** | CPC(클릭당), CPM(1,000노출당), CPS(판매당) 혼합 |
| **타겟팅** | 키워드 기반, 제품 카테고리 기반, 오디언스 기반 |
| **입찰 방식** | 수동 입찰, 자동 입찰 (스마트 비딩) |
| **대시보드** | Shopee Ads 대시보드 + 셀러 센터 통합 |
| **최소 예산** | 일일 예산 제한 설정 가능 (캠페인당 최소 $5~$10) |

## 2. AMS 광고 종류 상세

### 2.1 검색 광고 (Search Ads) — 키워드광고

검색 결과 페이지(Search Engine Results Page)에 노출되는 **키워드 기반 광고**. 사용자가 특정 키워드를 검색하면 광고 상품이 검색 결과 상단에 표시된다.

**운영 방식:**
- 키워드별 입찰가(Bid) 설정
- 광고 노출 순위 = 입찰가 × 품질 지수(Quality Score)
- 클릭 시에만 비용 발생 (CPC)

**특징:**
- 모든 광고 유형 중 가장 높은 노출 보장
- 구매 의도가 높은 사용자 타겟팅
- 키워드 리서치가 성패를 좌우

### 2.2 디스커버리 광고 (Discovery Ads)

Shopee 플랫폼 내 여러 섹션에 노출되는 **관심 기반 광고**. 사용자의 행동 데이터를 기반으로 관련성 높은 제품을 노출한다.

**노출 위치:**
- Shopee 홈페이지 내 Daily Discover
- Shopee Game 내 광고 영역
- 제품 상세 페이지 (You May Also Like)
- 주문/구매 관련 페이지
- 장바구니 페이지

**특징:**
- 검색 광고보다 상위 퍼널 (상품 발견 목적)
- 사용자 관심사 기반 타겟팅
- 검색 광고와 함께 사용하면 시너지 효과

### 2.3 샵 검색 광고 (Shop Search Ads)

검색 결과 페이지 최상단에 **샵(스토어) 자체를 광고**로 노출.

**표시 정보:**
- 샵 이름
- 태그라인 (설정 가능)
- 바우처 정보
- 인기 상품 목록

**입찰 방식:**
- 키워드 기반 입찰
- 입찰 키워드 예시: 헤어 케어 샵 → "샴푸" 키워드 입찰

### 2.4 디스플레이 광고 (Display Ads)

Shopee 홈페이지 상단 슬라이드 배너 위치에 노출되는 **이미지/영상 기반 광고**.

**과금 방식:**
- 주로 CPM(1,000노출당 비용) 기반
- 브랜드 인지도 향상 목적
- 제품 단위보다 브랜드/샵 단위 광고에 적합

### 2.5 Boost 광고 (자동 최적화)

Shopee가 **알고리즘으로 자동 최적화**해주는 올인원 광고 상품.

**자동화 범위:**
- 입찰가 자동 조정
- 광고 노출 위치 최적화
- 검색어 자동 매칭
- 예산 자동 분배

**장점:**
- 초보 셀러도 쉽게 시작 가능
- 관리 시간 최소화
- AI 기반 성과 최적화

### 2.6 Associate (CPS) — 어소시에이트/제휴 광고

**CPS(Cost Per Sale)** 방식으로 판매가 발생했을 때만 비용이 발생하는 **성과형 광고**.

**특징:**
- 인플루언서/제휴 파트너가 상품 홍보
- 판매 수수료만 지불 (리스크 최소화)
- ROAS 예측이 용이함
- 프로모션(할인, 무료배송, 바우처)과 함께 활용 시 효과 ↑

### 2.7 브랜드 광고 (Brand Ads)

대형 셀러/브랜드 전용 광고 상품. Shopee 홈페이지 프리미엄 지면 활용.

**종류:**
- 브랜드 배너 (홈페이지 최상단)
- 브랜드 페이지 (랜딩 페이지)
- 브랜드 챌린지 (게이미피케이션)

**적합 대상:**
- 월 $5,000+ 광고 예산 보유 셀러
- 브랜드 인지도 구축이 목적인 기업

## 4. 예산 설정 전략

### 4.1 예산 설정 기본 원칙

```
일일 예산 = (월 광고 예산) ÷ 30일
캠페인 예산 = 일일 예산 × 캠페인 수
```

**권장 예산 배분 (초보):**

| 단계 | 예산 비중 | 설명 |
|:-----|:---------|:------|
| 테스트 (1~2주) | 20% | 키워드/소재 테스트 |
| 최적화 (3~4주) | 30% | 효율 좋은 키워드 집중 |
| 확장 (5주~) | 50% | 성과 좋은 캠페인 스케일업 |

### 4.2 초보 셀러 예산 운영법 ($500~$1,000 테스트)

#### Phase 1: 테스트 (1~2주) — 예산: $200~$300

**목표:** 유효 키워드 발굴 + 기본 데이터 수집

**액션 플랜:**
1. **제품당 10~20개 키워드** 선정 (롱테일 키워드 중심)
2. **일일 예산:** 캠페인당 $5~$10
3. **입찰 전략:** 수동 입찰 (CPC $0.10~$0.30)
4. **측정 지표:** CTR, CPC, 노출 수

**체크리스트:**
- [ ] 경쟁사 광고 키워드 분석 (Shopee 검색창 자동완성 활용)
- [ ] 브랜드 키워드 + 제품명 키워드 분리
- [ ] 성과 좋은 키워드만 별도 캠페인으로 분리
- [ ] CTR 0.5% 미만 키워드 제외

#### Phase 2: 최적화 (3~4주) — 예산: $300~$400

**목표:** ROAS 300%+ 달성, 비효율 키워드 제거

**액션 플랜:**
1. **CTR 상위 20% 키워드** 입찰가 20% 인상
2. **전환 0건 키워드** 일시 중지
3. **Discovery Ads** 병행 시작
4. **자동 입찰 전환** 테스트

**체크리스트:**
- [ ] 네거티브 키워드 등록 (비효율 검색어 차단)
- [ ] 광고 시간대별 성과 분석
- [ ] 모바일 vs 데스크탑 성과 비교
- [ ] 제품 상세 페이지 최적화 (광고 랜딩)

#### Phase 3: 확장 (5주~) — 예산: $200~$300 (재투자)

**목표:** 수익성 유지하며 매출 스케일업

**액션 플랜:**
1. **성과 좋은 캠페인 예산 2배 증액**
2. **유사 타겟 확장** (Lookalike Audience)
3. **Boost 광고** 전환
4. **Associate Program** 병행

### 4.3 예산 관련 팁

| 구분 | 팁 |
|:-----|:----|
| **최소 예산** | 검색 광고 $5/일, 디스커버리 $5/일 → 총 $10/일부터 시작 |
| **예산 소진 속도** | 3시간 내 소진 시 → 키워드 수 줄이거나 입찰가 인하 |
| **예산 증액 타이밍** | ROAS 300% 이상 안정화 시 20~30%씩 증액 |
| **시즌 대비** | 메가 세일 2주 전부터 예산 2배 증액 |

## 5. 광고 성과 측정 KPI

### 5.1 핵심 KPI 정의

| KPI | 계산식 | 목표치 (초보) | 목표치 (고도화) |
|:----|:-------|:------------|:--------------|
| **CTR** | 클릭수 ÷ 노출수 × 100 | 0.5%~1.5% | 1.5%~3.0%+ |
| **CPC** | 총비용 ÷ 클릭수 | $0.10~$0.30 | $0.05~$0.15 |
| **CR (전환율)** | 주문수 ÷ 클릭수 × 100 | 3%~8% | 8%~15%+ |
| **ROAS** | 매출 ÷ 광고비 | 200%~300% | 400%~800%+ |
| **CPA** | 광고비 ÷ 주문수 | <$5 | <$2 |
| **Impressions** | 총 노출수 | 1,000+/일 | 10,000+/일 |
| **ACoS** | 광고비 ÷ 매출 × 100 | 30%~50% | 10%~25% |

### 5.2 ROAS 계산 예시

```
광고비: $500
매출: $2,000
ROAS = $2,000 ÷ $500 = 4.0 (400%)

ACoS = $500 ÷ $2,000 × 100 = 25%
```

### 5.3 마켓별 권장 KPI 기준

| 마켓 | 목표 ROAS | 목표 ACoS | 비고 |
|:-----|:---------|:---------|:-----|
| 싱가포르 | 300%+ | 30% 이하 | 높은 CPC, 높은 구매력 |
| 말레이시아 | 400%+ | 25% 이하 | 중간 CPC |
| 태국 | 500%+ | 20% 이하 | 낮은 CPC, 높은 전환율 |
| 베트남 | 400%+ | 25% 이하 | 낮은 CPC, 낮은 AOV |
| 필리핀 | 350%+ | 28% 이하 | 중간 CPC |
| 브라질 | 250%+ | 35% 이하 | 높은 CPC, 관세 이슈 |

## 6. AMS 대시보드 활용법

### 6.1 대시보드 주요 섹션

| 섹션 | 주요 확인 항목 | 활용 팁 |
|:-----|:------------|:-------|
| **Overview** | 총 노출, 클릭, 지출, ROAS | 일일 트렌드 체크 (7일/30일 비교) |
| **Campaigns** | 캠페인별 성과 | CTR 1% 미만 캠페인 점검 |
| **Keywords** | 키워드별 CTR, CPC, 전환 | 주 1회 키워드 성과 리뷰 |
| **Search Terms** | 실제 검색어 리포트 | 네거티브 키워드 발굴 |
| **Product Ads** | 제품별 광고 성과 | 베스트셀러 집중 광고 |
| **Performance Graph** | 시간대별 성과 | 최적 시간대 확인 |
| **Promotions** | 프로모션 연동 성과 | 할인 + 광고 시너지 분석 |

### 6.2 대시보드 리포트 활용 3단계

**1단계: 매일 — 스코어카드 확인 (5분)**

```
✅ 오늘 총 지출: $XX
✅ 오늘 ROAS: XXX%
✅ 상위 3개 키워드 CTR: X.X%
⚠️ 비정상 캠페인 확인 (CTR < 0.3% or CPC > $0.5)
```

**2단계: 주간 — 상세 분석 (30분)**

```
📊 이번 주 vs 지난 주 비교
📊 키워드 성과 Top 10 / Bottom 10
📊 캠페인별 예산 소진율
📊 Discovery Ads vs Search Ads 성과 비교
```

**3단계: 월간 — 전략 리뷰 (1시간)**

```
🎯 총 ROAS 목표 달성률
🎯 마켓별/카테고리별 성과
🎯 신규 키워드 테스트 결과
🎯 다음 달 예산 계획 수립
```

### 6.3 대시보드에서 발견하는 5가지 인사이트

| 인사이트 | 발견 방법 | 실행 액션 |
|:--------|:---------|:---------|
| 특정 시간대 광고비만 소진 | 시간대별 리포트 확인 | 비효율 시간대 입찰가 50% 인하 |
| 검색어와 제품 불일치 | Search Terms 리포트 | 해당 검색어 네거티브 등록 |
| A제품은 ROAS 높음, B제품은 낮음 | Product Ads 리포트 | B제품 중단 or 소재 변경 |
| 디스커버리 광고 CTR 급등 | Discovery 별도 분석 | 입찰가 유지하며 예산 증액 |
| 특정 키워드 CPC 상승 | 키워드 트렌드 확인 | 수동 입찰가 조정 or 대체 키워드 발굴 |

## 7. 초보 → 고도화 전략 로드맵

### 7.1 초보 단계 (1~3개월)

**목표:** 광고 시스템 이해, 기본 KPI 안정화

| 주차 | 할 일 | 체크포인트 |
|:----|:------|:----------|
| 1~2주 | 검색 광고 1개 캠페인 생성, 10~20개 키워드 테스트 | CTR 0.5%+ |
| 3~4주 | 성과 키워드 집중, 실적 부진 키워드 제거 | ROAS 200%+ |
| 5~6주 | Discovery Ads 추가, 자동 입찰 테스트 | ROAS 250%+ |
| 7~8주 | Associate Program 시작, 프로모션 연동 | 전체 ROAS 300%+ |
| 9~12주 | Budget scale-up, 신규 마켓 확장 | 월 $1,000+ 광고비 |

### 7.2 중급 단계 (3~6개월)

**목표:** 캠페인 자동화, ROAS 최적화

- **자동 입찰(Auto Bidding):** 전환 최적화 모드로 전환
- **A/B 테스트:** 광고 이미지, 카피, 키워드 세트 테스트
- **시즌 캠페인:** 9.9, 11.11, 12.12 전용 캠페인 준비
- **멀티 마켓:** 2~3개 마켓 동시 운영, 마켓별 전략 차별화
- **대시보드 자동화:** 정기 리포트 템플릿 구축

### 7.3 고도화 단계 (6개월+)

**목표:** 데이터 기반 전략, 리타겟팅, 브랜드 광고

#### 리타겟팅 전략

Shopee CPAS (Cross-Platform Advertising Solution)를 통한 Meta(Facebook/Instagram) 리타겟팅:

1. **Shopee CPAS 온보딩:**
   - Meta Business Manager 계정 생성
   - Shopee에 CPAS 요청 제출
   - 카탈로그 세그먼트 공유

2. **리타겟팅 캠페인 구성:**
   - **상품 조회자:** 지난 7일 내 조회자 (CTR 2~3배 ↑)
   - **장바구니 포기자:** 담고 구매 안 한 고객 (전환율 5~10배 ↑)
   - **구매 이력자:** 재구매 유도 (ROAS 500%+)

3. **퍼널별 메시지:**
   ```
   TOF (인지): 브랜드 스토리텔링, 신제품 소개
   MOF (고려): 제품 리뷰, 비교 콘텐츠
   BOF (전환): 할인 코드, 무료배송, 한정 타임 세일
   ```

#### 자동 입찰 고도화

| 입찰 전략 | 설명 | 적합 상황 |
|:---------|:-----|:---------|
| **수동 CPC** | 직접 입찰가 설정 | 초보, 소규모 테스트 |
| **자동 입찰 (클릭 최대화)** | 예산 내 최대 클릭 확보 | 데이터 수집 단계 |
| **자동 입찰 (전환 최대화)** | 예산 내 최대 전환 추구 | 전환 데이터 50+ 확보 시 |
| **타겟 ROAS** | 목표 ROAS 기반 자동 입찰 | ROAS 300%+ 안정화 시 |
| **타겟 CPA** | 목표 CPA 기반 자동 입찰 | CPA 안정화 필요 시 |

#### 고도화 핵심 전략

1. **데이터 기반 키워드 확장:**
   - Search Terms 리포트 → 신규 키워드 발굴
   - 경쟁사 광고 키워드 분석
   - 시즌별 트렌드 키워드 선점

2. **멀티 캠페인 구조 최적화:**
   ```
   캠페인 1: 브랜드 키워드 (높은 전환율)
   캠페인 2: 제품 키워드 (중간 전환율)
   캠페인 3: 카테고리 키워드 (상위 퍼널)
   캠페인 4: 경쟁사 키워드 (공격형)
   ```

3. **프로모션 + 광고 시너지:**
   - 할인 쿠폰 + 광고 동시 집행 → 전환율 2~3배 ↑
   - 무료배송 + 광고 → CTR 50% ↑
   - 바우처 + 광고 → AOV 20% ↑

4. **멀티 마켓 거버넌스:**
   - 마켓별 전담 캠페인 운영
   - 통화/언어/문화 차이 반영
   - 마켓별 KPI 별도 관리

## 9. 실전 체크리스트

### 캠페인 생성 전
- [ ] 제품 정보 완전하게 입력 (이미지, 설명, 가격, 재고)
- [ ] 키워드 리서치 완료 (10~20개 롱테일 키워드)
- [ ] 경쟁사 광고 분석 완료
- [ ] 마켓별 문화/언어 고려

### 캠페인 운영 중
- [ ] 매일 5분 대시보드 확인 (CTR, CPC, ROAS)
- [ ] 주 1회 키워드 성과 리뷰 (하위 20% 교체)
- [ ] 2주마다 신규 키워드 테스트
- [ ] 월 1회 전체 전략 리뷰

### 캠페인 최적화
- [ ] CTR < 0.5% 키워드 중단 or 소재 변경
- [ ] CPC > 평균 2배 키워드 입찰가 인하
- [ ] ROAS < 200% 캠페인 예산 축소
- [ ] Search Terms 리포트에서 네거티브 키워드 추가

### 고도화
- [ ] CPAS (Meta 리타겟팅) 설정 완료
- [ ] 자동 입찰 전환 (타겟 ROAS 모드)
- [ ] A/B 테스트 지속적 실행
- [ ] 시즌별 캠페인 사전 준비

## 10. 자주 묻는 질문 (FAQ)

**Q: 최소 광고 예산은 얼마인가요?**
A: 캠페인당 일일 $5부터 시작 가능합니다. 검색 광고 + 디스커버리 광고 병행 시 $10/일 권장합니다.

**Q: 광고 시작 후 언제부터 결과가 나오나요?**
A: 일반적으로 3~7일 후 데이터가 안정화되기 시작합니다. 최소 2주는 운영 후 판단하세요.

**Q: ROAS가 100% 미만이면 중단해야 하나요?**
A: 신규 제품의 경우 초반 ROAS가 낮을 수 있습니다. 2주 이상 ROAS < 100% 지속 시 중단 고려.

**Q: 키워드광고와 디스커버리광고 중 어떤 것이 더 효과적인가요?**
A: 목적에 따라 다릅니다. 즉시 전환이 목표면 키워드광고, 브랜드 인지도면 디스커버리광고. 병행 운영이 가장 효과적입니다.

**Q: 자동 입찰이 좋은가요, 수동 입찰이 좋은가요?**
A: 초보는 수동 입찰로 시작해 데이터를 쌓은 후, 전환 데이터 50+ 확보 시 자동 입찰로 전환하세요.

**Q: Associate Program은 어떻게 시작하나요?**
A: Shopee 셀러 센터 > 마케팅 > Associate Program에서 신청 가능합니다. CPS 기반이라 리스크가 적습니다.
