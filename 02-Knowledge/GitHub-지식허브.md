## 📊 경제 & 금융

### 1. OpenBB (66.7k ⭐)
**링크:** https://github.com/OpenBB-finance/OpenBB
**설명:** 금융 데이터 플랫폼 — "connect once, consume everywhere"
- Python/Excel/MCP/REST API 모두 지원
- 주식, 옵션, 채권, 환율, 암호화폐 등 전 자산군 데이터
- AI Agent 연동 가능 (Hermes에 접목 가능)
- **우리 활용:** Jongdari 배틀루프에 OpenBB 데이터 소스 추가 검토

### 2. TradingAgents (54.5k ⭐)
**링크:** https://github.com/TauricResearch/TradingAgents
**설명:** LLM 멀티에이전트 트레이딩 프레임워크
- Analyst → Researcher → Trader → Risk Manager 계층 구조
- GPT/Claude/Gemini/Grok/DeepSeek 등 모든 LLM 지원
- **우리 활용:** 우리 AI Council과 유사한 구조! 벤치마킹 포인트
  - 우리: 📊기술적/🔍뉴스모멘텀/📉리스크 3명
  - TradingAgents: 4단계 계층 + 백테스팅

### 3. Microsoft Qlib (41.4k ⭐)
**링크:** https://github.com/microsoft/qlib
**설명:** AI 퀀트 투자 플랫폼 — MS가 만든 AI 기반 퀀트
- ML 기반 팩터 마이닝 → 모델 학습 → 백테스팅 파이프라인
- RD-Agent: LLM으로 자동 팩터 발굴
- **우리 활용:** 엔터프라이즈급 AI 퀀트 시스템 구조 참고

### 4. AKShare (18.7k ⭐)
**링크:** https://github.com/akfamily/akshare
**설명:** 중국 금융 데이터 Python 라이브러리
- A주/선물/옵션/경제지표 데이터
- 원클릭 데이터 수집
- **우리 활용:** 중국 시장 분석 시 데이터 소스

### 5. FinanceToolkit (4.7k ⭐)
**링크:** https://github.com/JerBouma/FinanceToolkit
**설명:** 150+ 재무 비율 계산 툴킷
- PE/PB/ROE/ROA 등 모든 재무 비율 투명 계산
- 플랫폼별 차이 표준화
- **우리 활용:** Jongdari 종목 분석에 재무비율 추가

### 6. Maybe (54.1k ⭐)
**링크:** https://github.com/maybe-finance/maybe
**설명:** 개인 재무 관리 앱 (Ruby on Rails)
- 자산/부채/투자/예산 통합 관리
- **참고:** 더 이상 유지보수 안 됨 (AGPLv3)

## 🧠 사회 · 문화 · 과학

### 7. Mind-Expanding Books (13.7k ⭐)
**링크:** https://github.com/hackerkid/Mind-Expanding-Books
**설명:** 경제/철학/역사/과학 도서 큐레이션
**주요 도서:**
- 📘 Shoe Dog (Nike 창업자 필 나이트) — 창업과 도전
- 📘 Black Swan (나심 탈레브) — 불확실성과 투자
- 📘 Principles (레이 달리오) — 브릿지워터 CEO의 원칙
- 📘 Sapiens (유발 하라리) — 인류의 역사
- 📘 Thinking, Fast and Slow (대니얼 카너먼) — 행동경제학
- **우리 활용:** 위 도서들 위키에 요약 정리

### 8. Build Your Own X (498k ⭐)
**링크:** https://github.com/codecrafters-io/build-your-own-x
**설명:** 직접 만들어보며 배우기 — 30개 이상 기술 영역
- 직접 Git 만들기, DB 만들기, OS 만들기
- 20+ 프로그래밍 언어 튜토리얼
- **우리 활용:** 시스템 이해도 향상 → 더 나은 트레이딩 시스템 구축

### 9. OSSU Computer Science (203k ⭐)
**링크:** https://github.com/ossu/computer-science
**설명:** MIT/Stanford 수준의 무료 CS 학위 커리큘럼
- 알고리즘/자료구조/시스템/ML/AI 전 과정
- **우리 활용:** CS 기초가 필요한 경우 이 커리큘럼 참고

### 10. Free Programming Books (360k+ ⭐)
**링크:** https://github.com/EbookFoundation/free-programming-books
**설명:** 50+ 언어 무료 프로그래밍 서적
- 모든 분야의 무료 전자책 링크
- **우리 활용:** 필요시 한글/영어 책 검색

### 11. Coding Interview University (310k+ ⭐)
**링크:** https://github.com/jwasham/coding-interview-university
**설명:** 빅테크 면접 대비 학습 커리큘럼
- 8개월 코스 → Google/Amazon/Apple 입사
- 체계적인 알고리즘/자료구조 학습 로드맵

### 12. The Book of Secret Knowledge (218k ⭐)
**링크:** https://github.com/trimstray/the-book-of-secret-knowledge
**설명:** DevOps/보안/네트워크 백과사전
- CLI 원라이너, 해킹 도구, 보안 체크리스트
- 시스템 관리자 필수 지식 모음
- **우리 활용:** Hermes 인프라 보안 강화

### 13. Public APIs (330k+ ⭐)
**링크:** https://github.com/public-apis/public-apis
**설명:** 50개 카테고리 수백 개의 무료 공개 API
- 금융, 뉴스, 날씨, 지도, 음악 등
- **우리 활용:** 새로운 데이터 소스 발굴
