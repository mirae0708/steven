# Obsidian 추천 플러그인 및 설정 가이드

## 필수 플러그인
- Obsidian Git: GitHub와 실시간 동기화
- Dataview: 쿼리/집계/자동화
- Kanban: 프로젝트 보드
- Tasks: 할 일 관리
- Templater: 템플릿 자동화
- Advanced URI: 외부에서 파일 직접 열기
- Excalidraw: 다이어그램

## 추천 설정
- Obsidian Git: Auto commit 5분, Auto pull on startup, Auto push after commit
- Dataview: 기본값
- Kanban: 프로젝트별 보드 생성

## 설치 방법
1. Settings → Community plugins → Browse → 플러그인 검색 후 설치
2. 필요한 경우 플러그인별 설정 적용
