### 2.1 🇻🇳 베트남

#### 📊 핵심 지표

| 항목 | 내용 |

| **Shopee 시장 점유율** | **67.9%** 🥇 (압도적 1위) |
| **TikTok Shop 점유율** | **23.2%** 🥈 (성장률은 Shopee 추월) |
| **인터넷 사용자** | **6,730만 명** (인구의 **96.9%**) |
| **뷰티 이커머스** | 2022년 3.04억$ → 2026년 **6.44억$** (2배↑) |
| **소비 성향** | 가성비 + 프리미엄 양극화 |

#### ⚠️ 주요 리스크 — 특별소비세

- **2026년 화장품 특별소비세(Special Consumption Tax) 도입 예정**
- 세율: 품목별 상이 (화장품 대상 인상)
- 영향: 가격 경쟁력 약화 → 프리미엄 포지셔닝 전환 필요
- 대응 전략:
  - 프리미엄 라인 vs 보급형 라인 **이원화 전략**
  - 현지 생산/ODM 파트너십 검토
  - Shopee SLS 물류 활용한 비용 최적화

#### 📋 규제 사항 — 전자상거래 라이선스

- **전자상거래 라이선스(ECT)** 필요
  - 베트남 산업무역부에서 발급
  - 외국 기업도 신청 가능하나 절차 복잡
  - Shopee 입점 시 Shopee가 일부 대행 가능
- **수입 관세**: 화장품 20~30% (HS 코드별 상이)
- **화장품 공고번호 등록**: 베트남 식약처 승인 필수
- **라벨링 규정**: 베트남어 표기 의무

#### 📊 핵심 지표

| **인구** | **2.8억 명** — 동남아 최대, 세계 4위 |
| **무슬림 비중** | **87%** — 할랄 인증 **필수** |
| **모바일 중심** | 스마트폰 보급률 70%+, 모바일 쇼핑 주도 |
| **COD 선호** | 현금결제(Cash on Delivery) 비중 높음 |

#### ⚠️ 할랄 인증 — 필수 사항

- **2026년 기준 할랄 인증 의무화 확대**
  - 화장품, 뷰티 제품도 할랄 인증 대상 포함
  - BPOM(인도네시아 식약처) 등록 + 할랄 인증서 필요
- **금지 성분 체크**: 돈피 유래 성분, 알코올(일부), 동물 실험 제품
- **인증 소요 시간**: 3~6개월 (사전 준비 필수)
- **비용**: 제품당 약 $500~$2,000

#### ⚠️ TikTok Shop 규제 배경

| 일정 | 사건 |
|:----|:-----|
| **2023.09** | 인도네시아 정부, **소셜미디어+이커머스 분리 규제** 발표 |
| **2023.10** | TikTok Shop 인도네시아 **서비스 중단** |
| **2023.12** | ByteDance, **Tokopedia에 $1.5B 투자** + 합작법인 설립 |
| **2024~2026** | TikTok Shop Indonesia = **Tokopedia × TikTok 합작 체제** 재개 |
| **현재(2026)** | Shopee **1위 유지** 중, TikTok Shop 제한적 운영 |

> ✅ **핵심 통찰**: 인도네시아는 소셜커머스 규제의 **선례** — 타 국가에도 영향 가능  
> ✅ Shopee는 규제 리스크가 낮아 **안정적 플랫폼**으로 선호

### 2.3 🇹🇭 태국

#### 📊 핵심 지표

| 항목 | 내용 |
|:----|:------|
| **시장 규모** | ~**100억$** |
| **스마트폰 보급률** | **85%+** — 동남아 최고 수준 |
| **소비 성향** | **트렌드 민감도 높음**, 신상품 빠른 채택 |
| **라이브커머스** | Shopee Live 중심 **매우 활성화** |
| **배송 선호** | **즉시 배송/퀵커머스** 선호도 ↑ |

#### 🔥 트렌드 민감도

- 태국 소비자는 **신제품 조기 채택율**이 동남아 최고
- K-뷰티, K-팝, K-콘텐츠에 대한 **강한 선호**
- **숏폼 영상 + 라이브 시연**이 구매 결정에 큰 영향
- 시즌별/트렌드별 **빠른 제품 전환** 필요

#### 🚚 즉시 배송 선호

- 태국은 **"가격 경쟁 종료"** 선언 — 배송 속도로 승부
- **퀵커머스(Quick Commerce)**: 주문 후 2-4시간 내 배송 선호
- 방콕 등 대도시 중심 **현재 물류 거점** 확보가 경쟁력
- Shopee SLS(5-10일) vs 현지 FBS(1-2일) — **FBS 전환 검토**

#### 💡 진출 전략

| 전략 | 내용 |
|:----|:------|
| 🏪 **플랫폼** | Shopee + **Shopee Live** 집중 |
| ⏱️ **배송** | 현지 물류 거점 → **퀵커머스** 대응 |
| 📢 **마케팅** | 숏폼 콘텐츠 + 인플루언서 + 라이브커머스 |
| 🆕 **신상품** | **월 1회 이상** 신제품 론칭 (트렌드 민감 대응) |
| 🎯 **타겟** | 18-30대 MZ, K-컬처 팬덤 |
| 📋 **규제** | 태국 FDA 등록 필요 (일부 품목 간소화 가능) |

### 2.4 🇵🇭 필리핀

#### 📊 핵심 지표

| 항목 | 내용 |
|:----|:------|
| **시장 규모** | ~**80억$** |
| **인구** | **1.15억 명** — 동남아 2위 |
| **영어 사용** | 공용어 — 현지화 부담 **낮음** |
| **SNS 의존도** | 세계 최고 수준 — **KOL 마케팅 절대적** |
| **K-뷰티 수요** | 글로우·듀이·K더마 **3대 키워드** 중심 폭발적 |

#### 🔑 핵심 키워드: 글로우(Glow) · 듀이(Dewy) · K-더마(K-Derma)

| 키워드 | 설명 |
|:------|:------|
| **✨ 글로우(Glow)** | 광채 피부 — 하이라이터, 글로우 쿠션, 광채 메이크업 |
| **💧 듀이(Dewy)** | 촉촉한 피부 — 수분감, 물광 메이크업 |
| **🔬 K-더마(K-Derma)** | 한국 더마 화장품 — 니아신아마이드, 센텔라, 레티놀 등 |

> 이 3개 키워드는 필리핀 뷰티 마켓의 **검색량 1-3위**를 차지  
> K-뷰티 = 프리미엄 + 효과성에 대한 높은 신뢰도

#### 👑 KOL 마케팅 중요성

- **KOL(인플루언서) 마케팅이 구매 결정의 핵심 채널**
- K-팝 팬덤과 강한 연계 — BTS, 블랙핑크 관련 뷰티 제품 선호
- 추천 KOL 유형:
  - **마이크로 인플루언서** (1~10만 팔로워): 높은 신뢰도, 저렴한 비용
  - **뷰티 유튜버**: 제품 시연/리뷰에 강한 영향력
  - **K-뷰티 전문 인플루언서**: 한국 화장품 전문 리뷰어
- Shopee Live + KOL 협업 시 **전환율 2~3배 상승**

#### 💡 진출 전략

| 전략 | 내용 |
|:----|:------|
| 🏪 **플랫폼** | Shopee 우선 + TikTok Shop 병행 |
| 🏷️ **키워드** | **글로우·듀이·K더마** 중심 상품 기획 |
| 👑 **KOL** | 현지 K-뷰티 인플루언서 Pool 구축 필수 |

| 📦 **제품** | 가벼운 텍스처(쿠션, 선케어, 토너) 선호 |

#### 🏢 지주사 활용 전략
